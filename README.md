# Beyond Light

Interactive 3D scrollable journey through the universe built with React + Three.js.

Nine full-screen scenes: stars, spiral galaxy, solar system, time, spacetime curvature, wormhole tunnel, black hole, and multiverse — each filling the viewport with the camera flying between them on scroll.

## Stack

- React 19 + Vite
- Three.js + @react-three/fiber + @react-three/drei
- GitHub Actions CI/CD to GitHub Pages

## Dev

```bash
npm install
npm run dev
```
